Map2Check First Example

In this first example, we show how to apply Map2Check tool to identify a simple invalid free in the following C source code, called invalid_free.c 

                                #include <stdio.h>
                                #include <stdlib.h>

                                int main()
                                {
                                    int *a = malloc(12);
                                    int *b = malloc(12);
                                    int  c =  12;
                                    a = &c;
                                    b = a;
                                    
                                    free(b);
                                    free(a);
                                }

Map2Check should be called in the installation directory as follows: ./map2check invalid_free.c.
