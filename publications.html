Publications and Talks Using Map2Check

Publications

Below you can find a list of papers and talkers that either use or extend Map2Check. They are listed in chronological order.
If you have used or extended Map2Check and would like to have your paper listed here, please email map2check.tool@gmail.com).

  2. Rocha, H. O., Barreto, R. S., Cordeiro, L. C. Memory Management Test-Case Generation of C Programs using Bounded Model Checking. In 13th International Conference on Software Engineering and Formal Methods (SEFM), LNCS 9276, pp. 251-267, 2015.
  3. Rocha, H. O., Cordeiro L. C., Barreto, R. S. Hunting Memory Bugs in C Programs with Map2Check (Competition Contribution). In 22th Intl. Conf. on Tools and Algorithms for the Construction and Analysis of Systems (TACAS), LNCS 9636, pp. 934-937, 2016.
